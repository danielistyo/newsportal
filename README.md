# newsportal
This repo is using Vue-CLI to create headline news website. Due to Vuetify is still alpha version for supporting Vue 3, then I customized some Vuetify components to handle some cases.

## Tech Stack
- Vue 3
- Vuex
- Vue Router
- Typescript
- Vuetify

## Project setup
```
yarn install
yarn serve
```
