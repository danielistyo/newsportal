# newsportal

This repo is using Vue-CLI to create headline news website. 

## Tech Stack

- Vue 3
- Vuex
- Vue Router
- Vuetify

## prerequisite

- Node v18.16.1

## Project setup

```
yarn install
yarn serve
```

## Run Test

```
yarn test:unit
```
